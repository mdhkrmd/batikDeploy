Behind the App

The process of creating this application

Dataset

There are 5 classes of batik on this dataset (Kawung, Megamendung, Parang, Sekarjagad, Truntum) with total 2450 images

Model

We used EfficientNet V2-B0 to build the image classification model. 96% on validation accuracy and 94% on test accuracy

Back-end

We used FastAPI as the main web framework to create an API

Front-end

We used Bootstrap 5
